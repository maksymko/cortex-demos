Detailed Description

template<typename T>
class mock::WriteSink< T >

Write values into a sink.

Each subsequent write to the handled address will be stored in a vector, after being cast to type T.

This is useful for simulating outgoing transmission of some sort for UART, I2C etc. peripherals.
