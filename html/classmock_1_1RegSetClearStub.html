Detailed Description

IO Handler for the register with separate "set" and "clear" registers.

Implements set/clear/rw addresses for the same register.

Writing to "set" address sets the bits in the register that are set in the written value. All other bits are preserved.

Writing to "clear" address clears the bits in the register that are set in the written value. All other bits are preserved.

The main address has usual Read/Write access. All addresses read as a current value for the register.
